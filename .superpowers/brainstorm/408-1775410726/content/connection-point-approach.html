Where do relationship edges connect?

Three approaches for how edges attach to table nodes. Click an option to select it.

A

Table-center connections

Edges connect to the center of each table. Simplest, spec default. No attribute-level selection.

B

Border-snap connections with row highlight

Edge exits from the nearest side border. User optionally clicks a row to anchor the connection to that row's Y position — but the edge still snaps to the border edge.

C

Full attribute-row handles (Phase 5 + Phase 6 unified)

Each attribute row has explicit React Flow handles. Relationships AND FDs both use the same row-level handles. Does more now, Phase 6 FD work is smaller.
